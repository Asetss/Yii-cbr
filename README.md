Yii-cbr
=======
Виджет валюты для Yii.Валюта пишеться по стандарту ISO 4217

### Использование виджета:
-----
~~~
     $this->widget('CBRwidget', array('curs'=> array('EUR')));
~~~

### Использование несколько валюты:
-----
~~~
     $this->widget('CBRwidget', array('curs'=> array('EUR','USD')));
~~~
